I needed something that would auto-rate my Plex library from Tidal's popularity and listening ranks.

Script has GUI - connects to Plex and Tidal (authorized via web browser)

Has option to clear ALL Plex ratings

Allows rating Playlists, Artists and Albums

Allows you to save settings on Exit

Added option to not tag songs with identical names so hopefully no dupicates, it should pick the 1st album release.

Requirements: Python and pip

pip install PyQt6 plexapi requests tidalapi
